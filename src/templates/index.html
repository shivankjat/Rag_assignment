Welcome to the Text-to-Answer API

This API allows you to convert natural language questions and generate answers using advanced AI models.

Available Endpoints:

GET /train: Initiates the training process for the model.
  • Without parameters: Trains the model using the current database schema.
  • With parameters (optional):
    • csv: The path to a CSV file for additional training data
    • question: A specific question to train on
    • documentation: Additional documentation to include in training

Examples:

GET /train

This will train the model using the current database schema.

GET /train?csv=path/to/your/file.csv

This will train the model using both the database schema and the data from the specified CSV file.

POST /ask: Accepts a question and returns the answers and results.

Example:

POST /ask

Request body:


{
    "question": "What is the total revenue for 2023?"
}

This will return the answers and the results for the given question.
